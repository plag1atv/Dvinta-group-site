Какое оборудование мы аттестуем

ООО «ЦСМ Двинта» оказывает услуги по первичной и периодической аттестации испытательного оборудования. Наши специалисты помогут разобраться в возможности проведения первичной аттестации оборудования.

Перечень
Примеры оборудования
  • климатических камер;
  • термостатов;
  • прессов, испытательных машин, разрывных машин;
  • печей, плит, сушильных шкафов, холодильных камер;
  • оборудования для создания специфических условий испытаний;
  • оборудования для испытания грунтов, песка, земли;
  • оборудования для определения жесткости бетонной смеси;
  • оборудования для испытаний пищевой и нефтегазовой продукции.

Сроки
Какие сроки аттестации у нас?

Сроки выполнения аттестации испытательного оборудования определяются после анализа заявки на аттестацию и зависят от технического задания заказчика.

Важно: возможна аттестация оборудования на месте эксплуатации (выездные работы) — по согласованию.

Вопросы и ответы

Собрали ответы на самые частые вопросы по поверке средств измерений.

Испытательное оборудование (ИО) – технические устройства, предназначенные для воспроизведения условий испытаний при экспериментальном определении характеристик объектов.
Аттестация ИО — определение нормированных точностных характеристик испытательного оборудования, их соответствия требованиям нормативно-технической документации и установление пригодности этого оборудования к эксплуатации.
Оказываем услуги по первичной и периодической аттестации:
  • климатических камер;
  • термостатов;
  • прессов, испытательных машин, разрывных машин;
  • печей, плит, сушильных шкафов, холодильных камер;
  • оборудования для создания специфических условий испытаний;
  • оборудования для испытания грунтов, песка, земли;
  • оборудования для определения жесткости бетонной смеси;
  • оборудования для испытаний пищевой и нефтегазовой продукции.
По запросу возможна разработка методик аттестации испытательного оборудования.
Сроки выполнения аттестации определяются после анализа заявки на аттестацию и зависят от технического задания заказчика. Возможна аттестация оборудования на месте эксплуатации — по согласованию.
